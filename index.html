Crypt

A Simple Script That Quickly Encrypts Text Into An Unreadable Incoherent String Of Chars And Back Again.

Originally Made For The String Encrypter V2

Disclaimer:

crypt, Despite its Name, Is Not Secure. It Is Not To Be Used In Any Place Where Any Form Of Sensitive Data Is Being Transferred Or Stored. It Is Easy To Break And Can Be Broken Quickly Using A Simple Brute-Force Attack.

Usage:

Include the following in the head of your page's html code to use the publically hosted version. This will export a single function to the global scope of the window called crypt. It has no parameter validation, but to function correctly it must be called with the string to be encrypted, and the cipher text. It's output is rarely printable characters so it should be encoded in base64 or stored as a buffer.

<script src="https://giansinghsarao.github.io/crypt/crypt.min.js"></script>
